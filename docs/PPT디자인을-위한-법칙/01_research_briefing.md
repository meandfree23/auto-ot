## 핵심 인사이트 요약 (3~5줄)

PPT 디자인의 본질은 "꾸미는 것"이 아니라 **정보를 정리하는 과정**이다. 청중은 슬라이드를 읽지 않고 "본다" — 따라서 1슬라이드 1메시지, 텍스트 최소화, 비주얼 중심 구성이 핵심이다. 색상은 6:3:1 법칙(배경:본문:강조)으로 3가지 이내로 제한하고, 폰트는 고딕체 계열 2종 이하로 통일한다. 여백은 낭비가 아니라 전달력을 높이는 설계 도구이며, 슬라이드 면적의 15~40%는 비워두는 것이 프로의 기준이다. 일관성(폰트·컬러·마진·정렬)이 전문성을 만들고, 슬라이드 마스터가 그 일관성을 자동으로 보장한다.

## 섹션별 핵심 내용

### 1. PPT 디자인의 철학 — "읽는" 것이 아닌 "보는" 디자인

PPT는 보고서와 근본적으로 다르다. 보고서는 일대일로 읽히지만, 프레젠테이션은 일대다수로 "본다".

- **청중은 절대 읽지 않는다**: 슬라이드에 적힌 글을 정독하는 청중은 없다. 핵심 키워드만 쓰고, 나머지는 발표자의 말로 설명한다.
- **디자인의 본질은 "정리"**: LG전자 10년 차 인터랙션 디자이너 윤춘근은 "PPT 디자인은 예쁘게 꾸미는 것이 아니라 정보를 정리하는 과정"이라고 정의한다.
- **1슬라이드 1메시지 법칙**: 한 장에는 핵심 키포인트 1개만. 내용이 넘치면 폰트를 줄이는 것이 아니라 슬라이드를 나눈다.
- **3요소 배치 공식** (스킬텔러 채널): 임팩트 있는 이미지(아이캐처) + 흥미를 유발하는 제목 + 구체적 핵심 카피(서브 카피) 조합.
- **슬라이드는 "그림판"**: 말을 보조하는 시각 자료다. "보여주고 말로 설명한다"는 원칙 하나만 지켜도 퀄리티가 급상승한다.

### 2. 타이포그래피 — 가독성이 개성보다 먼저다

#### 폰트 선택 원칙

| 구분 | 규칙 | 이유 |
|------|------|------|
| 폰트 종류 | 최대 2가지 (제목용 1 + 본문용 1) | 3종 이상은 시각적 소음 발생 |
| 폰트 계열 | 화면·투사용은 고딕체(Sans-serif) | 명조체의 삐침(Serif)이 화면에서 번짐 |
| 추천 한글 폰트 | 프리텐다드, 나눔스퀘어, G마켓 산스 | 가독성 + 무료 사용 가능 |
| 피해야 할 폰트 | 굴림체, 캘리그라피체(본문 사용 시) | 가독성 저하, 조형성 낮음 |

**굴림체 금지**: 윤춘근 디자이너는 "굴림체는 일본 나루체를 강제 한글화한 서체로, 가독성이 떨어지고 조형적으로도 나쁘다"고 명시적으로 경고한다.

#### 폰트 크기 기준 (수치)

| 요소 | 최소 크기 | 권장 크기 |
|------|-----------|-----------|
| 슬라이드 제목 | 28~32pt | 36~44pt (Bold) |
| 본문 텍스트 | 18pt | 20~24pt |
| 라벨 / 캡션 | 12pt | 14~16pt |
| 출처 표기 | 12pt | 14pt |

> 핵심 숫자나 헤드라인은 본문보다 **2~3배 크게** 설정해 시선을 단번에 끌어야 한다 (예: 핵심 수치 60pt 이상, 본문 18~24pt).

#### 자간 · 행간 설정법 (헤이디 PPT 디자인 채널)

- **행간(줄 간격)**: 기본값 1.0배는 너무 좁다. **1.2~1.3배**로 넓혀야 가독성이 높아진다.
- **자간(글자 간격)**: 한글은 약간 좁히는 것이 그룹핑 효과를 준다. **0.5~2pt 좁게** 설정이 적당하다 (PPT 기준: 간격 → 좁게 → 0.8 정도).
- **설정 팁**: 행간·자간을 설정한 텍스트 상자를 우클릭 → "기본 텍스트 상자로 설정"하면 이후 모든 텍스트 상자에 자동 적용된다.

#### 시각적 계층 구조 (Typography Hierarchy)

굵기(Weight)와 크기(Size)의 대비로 제목 → 부제목 → 본문 순의 시선 흐름을 설계한다.

- 강조가 필요할 때 **새 폰트를 추가하지 말고** 크기·굵기·컬러를 변경한다.
- Level 1 (제목): 크고 굵게 → 3초 이내에 읽힌다
- Level 2 (부제목): 중간 크기, 보통 굵기 → 맥락 제공
- Level 3 (세부/출처): 작고 회색 → 필수 참고용

### 3. 색상 — 목적이 있어야 색이다

#### 6:3:1 법칙 (핵심 규칙)

```
배경색 60% : 본문/보조색 30% : 강조색 10%
```

윤춘근 디자이너가 강조하는 PPT 색감 분배의 핵심 법칙. 강조색 비율이 10%를 넘으면 더 이상 강조가 아니라 "소음(noise)"이 된다.

**적용 방법**:
1. 콘텐츠(이미지, 로고)에서 스포이트로 메인 컬러 추출
2. 배경은 해당 컬러 또는 무채색(흰색·회색·검정)으로 60% 할당
3. 본문·아이콘 등에 30%, 핵심 키워드·CTA에만 10% 적용

#### 색상 수 제한

- 전체 프레젠테이션에서 역할이 부여된 색상은 **3~4가지** 이하
- 색을 추출하기 어렵다면 **무채색(흰색·회색·검정)만으로도** 세련된 PPT가 완성된다

#### 색상 심리 기본

| 색상 | 연상 이미지 | 활용 상황 |
|------|-----------|-----------|
| 파랑 | 신뢰, 전문성, 차분함 | 비즈니스·기술 프레젠테이션 |
| 초록 | 성장, 균형, 안정 | 환경·금융·건강 주제 |
| 빨강 | 에너지, 긴급, 경고 | 핵심 지표 강조, 위험 표시 |
| 노랑 | 주목, 경고, 낙관 | 주의 환기 요소 |

#### 색상 조합 3가지 방법 (헤이디 PPT 디자인)

1. **로고/이미지에서 추출**: 브랜드나 주제 이미지의 스포이트 활용
2. **색 조합 사이트 활용**: Colorhunt.co, Adobe Color, Huemint (AI 기반)
3. **색상환 활용**: 유사색(안정적·차분함) / 보색(강렬·임팩트) / 단색(세련·미니멀)

### 4. 여백 (Whitespace) — 비워야 보인다

여백은 낭비된 공간이 아니라, 콘텐츠를 숨 쉬게 하고 핵심에 시선을 집중시키는 설계 도구다.

#### 수치 기준

| 상황 | 권장 여백 비율 |
|------|--------------|
| 일반 본문 슬라이드 | 슬라이드 면적의 **15~20%** 이상 비워두기 |
| 고급·프로페셔널 슬라이드 | **30~40%** (아마추어와 프로의 차이) |
| 타이틀 슬라이드 | **50%** 이상 |
| 가장자리 마진 | 상하좌우 **5~10%** 확보 |

> "좁은 폰트로 우겨넣지 말고 슬라이드를 나눠라." — 콘텐츠가 넘칠 때 폰트를 줄이는 것은 최악의 선택이다.

### 5. 레이아웃 & 이미지 — 구조가 메시지다

#### 그리드와 정렬

- **보이지 않는 12단 그리드** 기준으로 모든 요소를 정렬한다
- 텍스트, 이미지, 도형이 같은 기준선에 맞춰져야 "정돈된 느낌"이 만들어진다
- 5픽셀의 어긋남도 시청자에게 "뭔가 이상한데"라는 불편함을 준다
- 영어·한글 본문은 **좌측 정렬**이 가장 읽기 편하다

#### 히어로 이미지 (Hero Image) 활용

> "좋은 이미지 하나만으로 디자인의 80%가 완성된다." — 윤춘근 디자이너

- 슬라이드 면적의 **80% 이상**을 고해상도 이미지로 채우면 몰입도·신뢰도 급상승
- 이미지 해상도 기준: 최소 **1920×1080 (Full HD)** 이상
- 인위적이거나 연출된 사진보다 **자연스럽고 감성적인** 이미지가 더 효과적

**이미지 위 텍스트 가독성 확보 방법**:
- 이미지의 여백(비어있는 영역)에 텍스트 배치
- 텍스트 뒤에 **투명도 70~80%의 반투명 직사각형 레이어** 삽입
- 이미지 자체에 그라데이션·필터 처리

#### 이미지 선택 기준 5가지

1. 주제와 연관성
2. 분위기가 자연스러울 것
3. 고해상도 (Full HD 이상)
4. 강한 원색 이미지는 피할 것
5. 텍스트를 넣을 충분한 여백 확보

#### 8가지 공통 슬라이드 유형과 레이아웃 패턴

| 슬라이드 유형 | 핵심 규칙 |
|--------------|-----------|
| 타이틀 슬라이드 | 제목 36~44pt, 부제목 18~24pt, 최소한으로 |
| 섹션 구분 | 메인 브랜드 컬러 배경, 텍스트 최소화 |
| 투컬럼 | 좌우 상단·하단 정확히 정렬 |
| 데이터 + 인사이트 | 차트(60%) + 통찰 텍스트(40%), 인사이트가 더 중요 |
| 풀블리드 이미지 | 고해상도 필수, 반투명 텍스트 오버레이 |
| 비교 | 동일 구조, 내용만 다르게 |
| 목차/어젠다 | 현재 섹션 강조색으로 하이라이트 |
| 핵심 메시지 | 단 한 문장, 24~28pt, 강조색 |

### 6. 일관성 — 신뢰의 설계

> "색상과 폰트가 슬라이드마다 바뀌면, 청중은 발표 내용 자체의 전문성을 의심한다." — 크몽 10년 차 디자이너

- 첫 장부터 끝 장까지 동일한 폰트, 동일한 컬러 팔레트, 동일한 마진
- **슬라이드 마스터** 기능으로 가이드라인·서식·안내선을 고정하면 자동으로 일관성 유지
- 반복(Repetition)이 통일감을 만들고, 통일감이 브랜드 신뢰도를 만든다

---

### 7. 애니메이션 — 양념이지 주재료가 아니다

**사용해야 할 때**: 정보가 논리적 순서로 공개되어야 할 때만 (예: 단계적 프로세스, 그래프 데이터 순차 공개)

**사용해도 되는 효과**: Appear(즉시 표시), Fade(서서히 등장), Wipe(방향 전환)

**금지 효과**: Fly In, Bounce, Spin, Zoom, Swivel, Float — PowerPoint의 "Exciting" 카테고리에 있는 모든 효과

> "화려하게 날아다니는 애니메이션은 90년대 PPT 같은 느낌을 주고 전문성을 해친다."

## NotebookLM 분석 결과

NotebookLM 노트북 "PPT디자인법칙_리서치"에 YouTube 5편 + 전문 아티클 5편(총 10개 소스)을 입력하고 3가지 쿼리를 수행한 분석 결과 요약.

### 쿼리 1: PPT 디자인의 핵심 법칙과 원칙

5가지 카테고리로 귀결됨:
1. **철학** — "읽는" PPT가 아닌 "보는" PPT. 1슬라이드 1메시지.
2. **타이포그래피** — 고딕체 2종 이하, 행간 1.2~1.3배, 자간 0.5~2pt 좁게.
3. **색상** — 6:3:1 비율, 3~4가지 이하, 이미지·로고에서 스포이트 추출.
4. **레이아웃** — 여백 15~40%, 그리드 정렬, 히어로 이미지 80% 이상.
5. **일관성·완성도** — 슬라이드 마스터 활용, 애니메이션 최소화, 목차 제공.
