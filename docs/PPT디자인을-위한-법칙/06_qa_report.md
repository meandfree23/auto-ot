## 이슈 목록

### ⚠️ ISSUE-01 — 제목 크기 28pt 미달 (24개 슬라이드)

**기준:** 헤이디 법칙 — 제목 폰트는 28pt 이상

**실측:** 일반 콘텐츠 슬라이드 제목이 대부분 **24~26pt**로 설정됨

| 슬라이드 | 실측 크기 | 텍스트 |
|----------|----------|--------|
| 3 | 26pt | PPT는 보고서가 아니다 — 청중은 절대 읽지 않는다 |
| 5~7 | 24~26pt | 콘텐츠 슬라이드 제목 전반 |
| 9~14 | 24pt | 타이포그래피 섹션 전 슬라이드 |
| 16~20 | 24pt | 색상 섹션 전 슬라이드 |
| 22~24 | 24~26pt | 여백/레이아웃 섹션 |
| 26~28 | 24pt | 비주얼 섹션 |
| 30~32 | 24~26pt | 마무리 섹션 |

**영향도:** Medium — 가독성에 직접적 영향은 없으나 헤이디 기준 미충족
**권장 조치:** 콘텐츠 슬라이드 제목을 24pt → **28pt**로 상향 조정

**기준:** 헤이디 법칙 — 본문 텍스트 15pt 이상

**실측:** 인용구(Quote), 서브카피, 표 내부 레이블 등에 **14pt** 사용

| 슬라이드 | 실측 크기 | 텍스트 |
|----------|----------|--------|
