## 발견된 이슈

### ISSUE-01 (Medium) — 전체 슬라이드 여백 1cm 미달

**대상**: 32/32 슬라이드 전체
**규칙 근거**: design_rules.md §4 "상하좌우 최소 1cm씩"

**패턴 A — 섹션 레이블 상단 여백** (콘텐츠 슬라이드 24개)
- `TextBox 2` (섹션 번호: `#1`, `#2` 등): top **0.64cm** (기준 1cm 미달 0.36cm)
- `TextBox 3` (섹션명: `물리화학`, `지구과학` 등): top **0.64cm**, right **0.25cm**

**패턴 B — 우측/하단 통계 박스** (콘텐츠 슬라이드 대부분)
- 우측 통계 텍스트박스: right **0.25~0.51cm** (기준 미달)
- 하단 통계 텍스트박스: bottom **0.25cm** (기준 미달)

**패턴 C — 섹션 커버 데이터 박스** (섹션 커버 슬라이드)
- 섹션 번호 박스: left/top **0.76cm**, right/bottom **0.76cm**

**수정 방법**: 각 패턴별 공통 수치를 1.0cm로 일괄 상향. TextBox 2/3의 top을 0.64→1.0cm, 우측 통계박스 right를 0.25~0.51→1.0cm로 조정.

---

### ISSUE-02 (Low) — 슬라이드 32(결론) 메인 제목 28pt 미달

**대상**: 슬라이드 32 (결론/Q&A)
**현황**: `TextBox 10` ('얼음은 지금 우리에게 묻고 있다')이 `22pt` — 28pt 미달.
요약 불릿(TextBox 4)도 18pt로 설정되어 결론 슬라이드의 시각적 위계가 다른 콘텐츠 슬라이드 대비 약화됨.

**수정 방법**: TextBox 10의 핵심 메시지를 최소 28pt로 상향.

---

### ISSUE-03 (Low) — 색상 3색 체계 외 3가지 색 추가 사용

**대상**: 전체 슬라이드
**현황**:
- `#DC2626` (빨강): 9회 — 슬라이드 8 알베도 차트 경고 수치 표현
- `#D97706` (앰버 변형): 4회 — Accent `#F59E0B`와 유사 계열이나 별개 HEX 코드
- `#16A34A` (초록): 1회 — 단독 포인트 사용

**평가**: DC2626은 데이터 시각화(온도 상승/경고)에서 의미론적으로 정당하나 설계 원칙상 3색+중립 체계를 벗어남. D97706는 F59E0B의 변형으로 동일 계열이지만 별도 HEX 코드는 불일치.

**권장**: D97706를 F59E0B로 통일. DC2626과 16A34A 사용처를 검토해 최소화하거나 설계 팔레트에 명시적으로 추가.

## 우수 사항 (Notable Strengths)

- **인사이트형 제목 100%**: 25개 콘텐츠 슬라이드 전체에서 구체적 사실, 수치, 결론이 포함된 제목 사용. "분기별 매출 현황" 유형의 설명형 제목 0건.
- **섹션 커버 디자인 일관성**: 7개 섹션 커버 모두 Primary(#1E3A5F) 배경, Accent(#F59E0B) 섹션 번호, 흰색 본문 제목 구조로 통일.
- **타이포 위계 정밀**: 90pt(표지 영문) → 80pt(섹션 번호) → 50pt(표지 한국어) → 40pt(섹션 부제) → 28pt(콘텐츠 제목) → 22/18pt(소제목/본문) → 14pt(캡션) — 7단계 명확한 위계.
- **도식화 활용**: 32슬라이드 중 21슬라이드에서 카드형 박스(#F8FAFC, #2563EB fill) 활용. 나열형 텍스트만 있는 슬라이드 최소화.
- **단일 폰트**: 전체 32슬라이드, 모든 텍스트 요소에서 맑은 고딕만 사용 확인.
